# Photos Folder

This folder contains your wedding photos organized by category.

## Folder Structure

```
photos/
├── wedding/      # Wedding photos
├── engagement/   # Engagement photos
└── others/       # Other photos (user uploads go here)
```

## How to Add Photos

1. **Wedding Photos**: Add images to `photos/wedding/` folder
2. **Engagement Photos**: Add images to `photos/engagement/` folder
3. **Others**: Photos uploaded through the website will be saved here automatically

## Supported Image Formats

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)
- BMP (.bmp)

## Notes

- Photos are automatically sorted by modification date (newest first)
- Maximum file size: 10MB per image
- The website will automatically display all images from these folders
